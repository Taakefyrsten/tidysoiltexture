# Package index

## Classification

Classify sand/silt/clay fractions to USDA texture classes

- [`classify_texture()`](https://taakefyrsten.github.io/tidysoiltexture/reference/classify_texture.md)
  : Classify soil texture from sand, silt, and clay percentages

## Visualisation

Build ggplot2-based texture triangle plots

- [`gg_texture_triangle()`](https://taakefyrsten.github.io/tidysoiltexture/reference/gg_texture_triangle.md)
  : Plot a soil texture triangle with sample points
- [`texture_surface()`](https://taakefyrsten.github.io/tidysoiltexture/reference/texture_surface.md)
  : Prepare a surface for interpolated rendering on a texture triangle
- [`geom_texture_contour()`](https://taakefyrsten.github.io/tidysoiltexture/reference/geom_texture_contour.md)
  : Add iso-contour lines to a texture triangle plot
- [`texture_label_style()`](https://taakefyrsten.github.io/tidysoiltexture/reference/texture_label_style.md)
  : Styling options for text elements in the texture triangle

## Data

Built-in reference data

- [`usda_texture_classes`](https://taakefyrsten.github.io/tidysoiltexture/reference/usda_texture_classes.md)
  : USDA soil texture class polygon vertices
